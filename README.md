## Summary
A node.js module to braid or aggregate more than one RSS feed (file or url) into a single feed and process via specified plugins.

## Installation
```
$ git clone git@github.com:KQED/rss-braider.git
$ cd rss-braider
$ npm install
```
## Test
`npm test`

## Examples
```
$ cd examples
$ node simple.js
$ node plugins.js
```
